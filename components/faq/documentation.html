JTheme FAQ Component Documentation

The FAQ component is a reusable, modular element for creating and displaying Frequently Asked Questions in WordPress sites built with the JTheme theme. It's fully customizable, responsive, and can be used in multiple places across your site.

Quick Start

There are three main ways to use the FAQ component:

  1. Using a shortcode in your content
  2. Calling a function in a template file
  3. Using the saved FAQ items from the admin area

1. Using Shortcode

The simplest way to use the FAQ component is with a shortcode in your WordPress content:

[jthem_faq title="سوالات متداول" subtitle="پاسخ به سوالات رایج شما"]

This will display all FAQ items that you've created in the admin area.

If you want to display specific FAQ items, use the items attribute with comma-separated IDs:

[jthem_faq items="1,2,3" title="سوالات متداول"]
Tip: You can find the ID of each FAQ item in the admin interface at Appearance → سوالات متداول.

2. Using Function in Templates

For more control, you can use the PHP function in your template files:

<?php
// Basic usage with items from the FAQ admin
jthem_faq([
    'title' => 'سوالات متداول',
    'subtitle' => 'پاسخ به سوالات رایج شما',
]);
?>

Or create custom items directly in your code:

<?php
// Custom items defined in code
$faq_items = [
    [
        'question' => 'سوال اول',
        'answer' => 'پاسخ اول',
        'is_open' => true // This item will be open by default
    ],
    [
        'question' => 'سوال دوم',
        'answer' => 'پاسخ دوم',
    ],
];

jthem_faq([
    'title' => 'سوالات متداول',
    'items' => $faq_items,
]);
?>

Admin Interface

The FAQ component comes with a built-in admin interface at Appearance → سوالات متداول. Here you can create, edit, and manage all your FAQ items. These items can then be shown on your site using either shortcodes or template functions.

The admin interface allows you to:

  • Create unlimited FAQ items
  • Use a rich text editor for answers (supports formatting, links, etc.)
  • Set which items should be open by default
  • Get the ID of each item to use in shortcodes

Customization Options

The FAQ component is highly customizable. Here are all the parameters you can use:

Parameter Type Default Description
title string 'سوالات متداول' The title of the FAQ section
subtitle string '' Optional subtitle for the FAQ section
items array [] Array of FAQ items or comma-separated list of IDs in shortcode
container_class string 'faq-container my-12' CSS classes for the container
accordion_type string 'single' 'single' or 'multiple' (allow multiple panels open)
icon_open string 'plus' Icon to show when panel is closed. Options: 'plus', 'chevron-down'
icon_close string 'minus' Icon to show when panel is open. Options: 'minus', 'chevron-up'
custom_id string '' Custom ID for the FAQ container

Advanced Example with All Options

<?php
// Advanced usage with all options
jthem_faq([
    'title' => 'راهنمای خرید',
    'subtitle' => 'همه آنچه برای خرید راحت نیاز دارید',
    'items' => $faq_items, // Array of FAQ items
    'container_class' => 'faq-container my-8 max-w-3xl mx-auto bg-gray-100 p-6 rounded-xl',
    'accordion_type' => 'multiple', // Allow multiple panels to be open at once
    'icon_open' => 'chevron-down',
    'icon_close' => 'chevron-up',
    'custom_id' => 'purchase-guide-faq',
]);
?>

FAQ Item Format

Each FAQ item has the following format:

[
    'question' => 'Your question here?',
    'answer' => 'Your answer here. Can include HTML markup.',
    'is_open' => true/false // Optional, default is false
]

Getting FAQ HTML as a String

If you need to get the FAQ HTML as a string (for example, to modify it or use it in a custom location), you can use the jthem_get_faq() function: 

<?php
$faq_html = jthem_get_faq([
    'title' => 'سوالات متداول',
    'items' => $faq_items,
]);

// Now you can use $faq_html however you want
echo $faq_html;
?>

Styling Guide

The FAQ component uses Tailwind CSS for styling and is fully responsive. If you want to customize the appearance, you can add your own CSS classes or override the default styles.

The main CSS classes you can target are:

  • .faq-container - The main container
  • .faq-header - The title and subtitle section
  • .faq-items - The container for all FAQ items
  • .faq-item - Each individual FAQ item
  • .faq-question - The question button
  • .faq-answer - The answer content
  • .faq-icon - The toggle icon
Tip: You can add your own classes to the container using the container_class parameter.

Troubleshooting

Common Issue: If the FAQ component isn't displaying correctly, make sure the components loader is included in your theme's functions.php file.

Common issues and solutions:

  • FAQ doesn't appear - Check that the component is properly loaded in functions.php
  • Styling issues - Make sure Tailwind CSS is loaded on your page
  • JavaScript not working - Check for JavaScript errors in the browser console
  • Multiple FAQs on one page - Each FAQ gets a unique ID, so multiple instances should work fine
